# Python Assignment

This repository contains Python scripts for Task 1 and Task 2.

## Task 1: Basic Arithmetic Operations
This program:
- Takes two numbers as input from the user.
- Performs four basic mathematical operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Displays the result of each operation on the screen.

This task helps understand user input, numeric functions, and arithmetic operators in Python.

## Task 2: Name Concatenation and Greeting Message
This program:
- Takes the user's first name and last name as input.
- Concatenates them to form the full name.
- Displays a personalized greeting message using the full name.

This task demonstrates string input, concatenation, and output formatting in Python.

## How to Run
Use the following commands in the terminal:

python task1.py  
python task2.py
